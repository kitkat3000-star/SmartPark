# SmartPark

SmartPark is a parking management system for establishing efficiency within parking lots.
This consists of: 
1. An OCR system - PIR sensors, RaspberryPi, FTP server, TailScale VPN, Anaconda, Cronjob in linux, Pytesseract, ImageAI
2. An Android mobile app - Android Studio, NodeJS
3. Bluetooth BLE beacons - Aruba Beacons
4. Firebase for database
